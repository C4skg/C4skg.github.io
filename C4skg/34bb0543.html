    • 彩蛋

      提示备份文件，所以访问 www.zip，下载下来压缩包，解压后查看 index.php.bak ,得到彩蛋的一部分

  2. 我Flag呢？

    • 彩蛋

      打开网页开发者工具，看控制台，根据提示输入函数 giveMeEgg() 得到一部分 flag

  4. 作业管理系统

    首先是登录系统，看下网页源代码，他给了提示，直接告诉你了账号密码均为 admin

    登录进去到了主页，主要功能点为 上传文件 、创建文件 和 查看文件 还有 备份

    直接看创建文件，创建一个 php 文件，然后进行编辑即可

  5. 彩蛋

    将以上四题得到的彩蛋进行拼接即可

  6. Ping

    查看源码发现做了前端验证，直接使用 hackbar 代替发包

  7. Vim yyds

    猜测是 vim 备份文件泄露，访问 .index.php.swp 下载备份文件

    使用命令恢复 swp 文件得到源码

    查看关键部分代码

    使用 POST 传入 password 后直接执行命令即可

  8. PHP是世界上最好的语言！！

    右键查看网页源代码，分析 js 代码，可看到最后一个请求体中的 type 为 run ，且第二个参数名为 php_code ，猜测用于执行命令

    使用 hackbar 进行请求执行命令即可

  9. 导弹迷踪

    先故意死一次，得到死亡后的提示为 crashed

    查看网页源代码，寻找链接的 js 库，在 game.js 文件中，找到对应关键词，并在其附近看到 finish 对应对象的返回信息

  10. 1zjs

    查看网页源代码发现就一个 js 文件，直接搜索关键词 flag ，f1ag 等

    访问 /f@k3f1ag.php

    发现是 JSFuck，直接复制后放到控制台中运行，得到 flag

  11. Http pro max plus

    完整 http 请求包如下

    访问后得到提示，让访问 /wtfwtfwtfwtf.php

    该页面处有三个按钮，但是都没什么用，最后查看网页源代码，发现了注释

    访问 /sejishikong.php，得到 flag

  12. 就当无事发生

    访问探姬博客，看见第一篇博客文章就是题目相关，但是没东西，看了题目给的链接

    猜测使用 github 进行版本管理，因此到 github 搜索 ProbiusOfficial，得到以题目连接为库名的 github 库

    查看提交历史，发现第二个提交记录位成功，猜测为脱敏前的文件

    查看该提交记录，找到 2023/04/29/什么是flag呢/index.html 文件，得到 flag

  13. Flag点击就送！

    根据提交特征和指纹识别，猜测是 Flask 框架的 SSTI ，但是进行注入的时候，发现无法实现

    猜测题目代码并不是典型的 SSTI 代码

    且访问 flag 后提示不是管理员不给 flag ，猜测是 session 伪造

    拿到自身 session 并解码后，得到解码后的 session 内容

    但是没有注入，无法取得密钥，猜测题目是让你猜密钥，因此猜测密钥是 LitCTF

    将浏览器中的 session 值替换为伪造后的密钥，再次访问 /flag ，得到 flag

    dbg 下断点后输入任意长度为 33 的字符串后于断点处暂停，并可以得到置换后的表
